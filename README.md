# E-Commerce Shopping Website

## Description
This project is a full-stack E-Commerce Shopping Website developed using HTML, CSS, JavaScript, PHP, and MySQL. The website allows users to browse products, view product details, create an account, log in, and add items to their cart. The goal of this project was to understand how front-end and back-end technologies work together to build a functional web application.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Features
- User Registration and Login Authentication
- Secure Session Handling
- Product Listing and Category Pages
- Individual Product Detail Pages
- Add to Cart Functionality
- Dynamic My Account Page
- User-specific Cart Data Stored in MySQL
- Responsive Layout for Different Screen Sizes

## Project Structure
The project is organized into multiple folders for better structure:

- **blush, lip_oil, foundation, powder, mascara, eyeshadow**  
  Contains product pages for different makeup categories.

- **images/**  
  Stores all product images, banners, icons, and media files used across the website.

- **PHP Files**  
  Handles backend functionality such as user login, registration, session management, and cart operations.

- **HTML Files**  
  Used to build the layout and structure of the website pages.

- **CSS Files**  
  Controls the design, styling, and responsiveness of the website.

- **JavaScript Files**  
  Adds interactivity and dynamic behavior to the user interface.

## How to Run the Project
1. Install **XAMPP** on your system.
2. Move the project folder to the **htdocs** directory.
3. Start **Apache** and **MySQL** from the XAMPP Control Panel.
4. Import the project database into **phpMyAdmin**.
5. Open the browser and run the project using: http://localhost/rarebeauty
