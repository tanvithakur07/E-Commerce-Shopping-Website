# E-Commerce Shopping Website

## Description
This is a full-stack e-commerce web application developed using HTML, CSS, JavaScript, PHP, and MySQL. The project includes user authentication, product management, and cart functionality.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Features
- User registration and login
- Session handling
- Add to cart
- Dynamic product display
- My Account page

## How to Run the Project
1. Install XAMPP
2. Move project folder to htdocs
3. Import database.sql into phpMyAdmin
4. Start Apache & MySQL
5. Open localhost/project-name
